# 5-Step Multi-Step Application Form

A sleek, responsive, and user-friendly 5-step multi-step application form designed to streamline data collection with a seamless user experience. 

## 🚀 Features

* **Progress Tracking:** Visual step indicator to show the user's current progress.
* **Smooth Navigation:** Seamless transitions between steps (Next/Previous).
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
* **Clean UI:** Modern design focusing on readability and accessibility.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript, ReactJS
